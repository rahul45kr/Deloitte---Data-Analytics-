# Deloitte---Data-Analytics-
📊 Super Store Sales Dashboard (Power BI)
📝 Project Overview
This project presents an interactive Sales Dashboard built using Power BI, inspired by the Deloitte Data Analytics Virtual Internship.
The dashboard provides insights into sales performance, profit trends, customer segments, and regional analysis, helping businesses make data-driven decisions.

🎯 Objectives


Analyze sales and profit performance


Identify high-performing products and regions


Understand customer segments and behavior


Create an interactive and visually appealing dashboard



🛠️ Tools & Technologies


Power BI (Data Visualization & Dashboard)


Microsoft Excel / CSV (Dataset)


Data Cleaning & Transformation (Power Query)



📂 Dataset Description
The dataset includes:


Order Date, Ship Date


Region, State, City


Category & Sub-Category


Sales, Quantity, Profit


Customer Segment



📊 Dashboard Features
🔹 Key KPIs


Total Sales


Total Profit


Total Orders


Profit Margin


🔹 Visualizations


Sales by Region (Map / Bar Chart)


Profit by Category & Sub-Category


Monthly Sales Trend (Line Chart)


Sales by Customer Segment


Top-Selling Products


🔹 Filters (Slicers)


Region


Category


Segment


Date Range



📈 Key Insights


Certain regions generate higher sales but lower profit


Technology category shows strong profitability


Seasonal trends observed in monthly sales


Corporate segment contributes significantly to revenue
